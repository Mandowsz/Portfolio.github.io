# Portfolio.github.io

## How to Run This Portfolio Locally

### Option 1: Open Directly
1. Go to your project folder.
2. Double-click `index.html` (or right-click and choose "Open with" → your browser).

### Option 2: Use a Local Server (Recommended)
1. Make sure you have [Node.js](https://nodejs.org/) installed.
2. Open a terminal/command prompt in your project folder.
3. Install `http-server` globally (only once):
   ```sh
   npm install -g http-server
   ```
4. Start the server:
   ```sh
   http-server -p 8080
   ```
5. Open your browser and go to [http://localhost:8080](http://localhost:8080)

---

Your portfolio is now ready to use and share!
